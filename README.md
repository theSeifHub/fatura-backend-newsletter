# Fatura Backend Problem: Newsletter
Fatura Hackathon, Backend problem: Newsletter API using Nodejs

1. Clone repo on your machine
2. Run `npm install`
3. Run `npm start`
4. Go to `http://localhost:3000/`, preferably on Postman
5. Endpoints:
    - POST `/subscriptions`
    - GET `/issues/:subscriberId`
    - POST `/issues `
